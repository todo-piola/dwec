# Gato Factos

### Esta app realiza una consulta a la API catfact.ninja

Se trata de una API gratuita que proporciona información sobre gatos.
En concreto, devuelve 3 factos aleatorios sobre gatos cada vez que se realiza una consulta.

## Tecnologías utilizadas

- React
- Vite
- CSS
- API catfact.ninja
